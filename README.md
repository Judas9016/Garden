# Campo de Flores

Un campo de flores 3D interactivo hecho con Three.js.

## Atajos de teclado

| Tecla | Acción |
|-------|--------|
| `D` | Acelerar ciclo día/noche (mantener presionado) |
| `P` | Pausa / reanudar (modo foto) |
| `R` | Activar / desactivar lluvia |

## Controles

- **Arrastrar** con mouse/dedo para rotar la cámara
- **Rueda** para zoom
- **Táctil** — pellizcar para zoom, arrastrar para rotar
